# Projeto 01 - Blink LED

## 🎯 Objetivo
Fazer um LED piscar utilizando o microcontrolador Arduino UNO.

## 🔧 Hardware necessário
- Arduino UNO
- 1 LED (qualquer cor)
- 1 Resistor 220 Ω
- Protoboard e jumpers

## 📜 Esquema
O LED deve ser conectado ao **pino digital 12** do Arduino através do resistor de 220 Ω.

## 💻 Código
O código está disponível na pasta `firmware/blink/blink.ino`.

## 🚀 Como usar
1. Abra o Arduino IDE
2. Carregue o código para o Arduino UNO
3. Observe o LED piscando a cada 0,5 segundos
